#  Programação Orientada a Objetos (POO) - Faculdade IMES

Este repositório contém os exercícios e projetos desenvolvidos durante a disciplina de **Programação Orientada a Objetos (POO)** do curso de **Análise e Desenvolvimento de Sistemas** da **Faculdade IMES**.

---

##  Objetivo
Aplicar os principais conceitos da **Programação Orientada a Objetos (POO)** utilizando a linguagem **Java**, desenvolvendo a lógica e as boas práticas de codificação voltadas ao paradigma orientado a objetos.

---

##  Conceitos estudados
Durante o curso, foram abordados os seguintes tópicos:

-  Classes e Objetos  
-  Encapsulamento (Getters e Setters)  
-  Construtores  
-  Herança e Polimorfismo  
-  Sobrecarga e Sobrescrita de métodos  
-  Modificadores de acesso  
-  Pacotes e Organização de código  
-  Coleções (List, Set)  
-  Manipulação de arquivos e entrada de dados  
-  Tratamento de exceções  

---

##  Tecnologias utilizadas
- **Java JDK 17+**
- **VS Code / IntelliJ IDEA**
- **Git e GitHub**

 Aprendizado pessoal

Durante o desenvolvimento das atividades, aprendi na prática como aplicar os pilares da POO, organizar melhor o código em classes e métodos, e desenvolver soluções mais estruturadas e reutilizáveis.Na Faculdade foi a base do meu aprendizado em java Poo
